mini-auth / manual test console

Test email OTP and passkey flows from one page

A lightweight browser UI for manually exercising the server APIs with real requests and real WebAuthn ceremonies.

Setup

Load this page from any static server, then start mini-auth on another origin with --issuer set to the auth server origin and --origin set to the current page origin shown below.

If your Auth Server is not at http://127.0.0.1:7777, open this page with ?sdk-origin=https://your-auth-origin.

Start Auth Server

mini-auth start ... --origin http://127.0.0.1:8080
